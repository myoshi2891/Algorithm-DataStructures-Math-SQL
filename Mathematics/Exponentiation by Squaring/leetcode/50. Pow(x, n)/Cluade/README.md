#### **数学的理由**

- **解決策**: `x^5 = x × x^4` として「1つ取り出す」
- **効果**: `n-1`は必ず偶数になり、次のステップで分割可能

#### **具体例: `2^7`の計算**
```
❌ 単純計算: 2×2×2×2×2×2×2 (7回の掛け算)

```

### **➖ 負の指数の処理も明確になりました**

#### **数学的変換**
- **定義**: `x^(-n) = 1/x^n = (1/x)^n`
- **変換**: `2^(-3) = (1/2)^3 = 0.5^3`
- **利点**: 同じアルゴリズムで処理可能
