## 📊 主な機能

1. **インタラクティブデモ**
    - 任意の文字列とパターンを入力して実行可能
    - リアルタイムでDPテーブルを生成・表示

2. **視覚的なDPテーブル**
    - 緑色（True）: マッチする状態
    - 赤色（False）: マッチしない状態
    - ステップバイステップの処理過程を表示

3. **アルゴリズムフロー図**
    - 4つの主要ステップを視覚的に表現
    - 処理の流れを直感的に理解

4. **具体例での詳細解析**
    - `s="aa", p="*"` の完全な処理過程
    - `s="adceb", p="*a*b*"` のインタラクティブ解析

## 🔍 技術的特徴

1. **'\*'の処理ロジック**
    - `dp[i][j-1]`: 空文字列として解釈
    - `dp[i-1][j]`: 1文字以上として解釈
    - 論理和による柔軟な対応

2. **計算量の視覚化**
    - 時間計算量 O(m×n) の説明
    - 空間計算量の最適化提案

3. **実装の核心部分**
    - TypeScriptコードの重要箇所をハイライト
    - 各条件分岐の意味を詳細解説
