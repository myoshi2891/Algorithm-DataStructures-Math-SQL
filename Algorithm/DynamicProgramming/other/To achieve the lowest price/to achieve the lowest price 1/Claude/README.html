📐 漸化式の説明

dp[i] = min(dp[i-1] + a, dp[i-2] + b)

意味:

  • dp[i-1] + a: (i-1)個まで最安で買って、1個追加
  • dp[i-2] + b: (i-2)個まで最安で買って、2個追加
